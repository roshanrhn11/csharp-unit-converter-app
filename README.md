\# C# Unit Converter Application



A desktop Unit Converter application developed using C# and .NET Windows Forms.  

This project demonstrates GUI development, event-driven programming, and basic software engineering concepts.



\## Technologies Used



\- C#

\- .NET

\- Windows Forms

\- Visual Studio



\## Features



\- Length conversion

\- Weight conversion

\- Temperature conversion

\- User-friendly graphical interface

\- Input validation



\## Skills Demonstrated



\- Object-Oriented Programming (OOP)

\- Event-driven programming

\- GUI application development

\- Exception handling

\- C# application development



\## Development Environment



\- Visual Studio 2022

\- .NET Framework

\- Windows Forms
